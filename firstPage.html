1. La programación web implica el uso de lenguajes como HTML, CSS, y JavaScript para crear sitios y aplicaciones web. Combina el diseño de interfaces visuales con la lógica del servidor y la interacción con bases de datos. El objetivo es desarrollar experiencias eficientes, interactivas y accesibles para los usuarios de la web.

  2. HTML (HyperText Markup Language) es el lenguaje estándar para estructurar y mostrar contenido en la web. Utiliza etiquetas para organizar el contenido, como texto, imágenes y enlaces. Es esencial para crear la estructura básica de una página web

  3. CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para describir la presentación visual de un documento HTML. Define cómo se verá el contenido, controlando aspectos como colores, fuentes y diseño de la página. Ayuda a mejorar la estética y la experiencia del usuario.

  4. JavaScript es un lenguaje de programación que permite la creación de contenido interactivo y dinámico en páginas web. Con JavaScript, los desarrolladores pueden agregar funciones como animaciones, formularios interactivos y actualizaciones de contenido sin recargar la página. Es clave para mejorar la interactividad del usuario.

  5. Python es un lenguaje de programación versátil y fácil de aprender, conocido por su sintaxis clara y sencilla. Se utiliza en una amplia variedad de aplicaciones, desde el desarrollo web hasta la inteligencia artificial. Python es altamente valorado por su capacidad para resolver problemas complejos con menos líneas de código.
